#  PartyPlanner: Sistema de Gestión de Eventos Sociales

Este repositorio contiene el código fuente y la documentación oficial del sistema web "PartyPlanner", desarrollado como proyecto académico de Ingeniería de Software. Su objetivo es facilitar la organización de eventos mediante la automatización del registro de invitados, control de aforo y confirmación de asistencia (RSVP).

##  Estructura del Repositorio

El proyecto está organizado bajo los estándares de desarrollo de software, separando el código fuente de la documentación técnica y las evidencias de calidad:

* `docs/` → Contiene toda la documentación técnica y manuales del sistema.
* `evidencias/pruebas/` → Capturas de pantalla y logs de los resultados de QA.
* `src/` → Código fuente del sistema (Frontend y Backend).
* `diagramas/` → Modelado del sistema (Casos de Uso, Diagrama ER).
